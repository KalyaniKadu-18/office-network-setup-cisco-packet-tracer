# Office Network Setup Using Cisco Packet Tracer

## Overview
Designed and configured a small office network using Cisco Packet Tracer.

## Devices Used
- Router 2911
- Switch 2960
- 5 PCs
- 1 Server

## Features
- IPv4 Addressing
- DHCP Configuration
- Router Configuration
- Switch Configuration
- Network Connectivity Testing

## Testing
Verified communication using ping between:
- PC and Router
- PC and Server
- PC and PC

## Skills
- Cisco Packet Tracer
- TCP/IP
- DHCP
- LAN Configuration
- Network Troubleshooting
